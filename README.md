# Advanced_Quiz_Game_with_Score_and_Hints_using_Python
An interactive command-line Quiz Game built with Python that uses recursion instead of loops, featuring score tracking, hints, retry functionality, and dynamic performance feedback.

Project Purpose

This project was developed to demonstrate recursion through a practical quiz application. It showcases how recursive function calls can replace loops while providing an engaging user experience with scoring, hints, and performance feedback.

Features Overview
Recursive quiz navigation without using loops
Questions stored using Python lists and dictionaries
Hint system for incorrect answers
Retry functionality with one additional attempt
Automatic score calculation
Performance feedback based on quiz results
